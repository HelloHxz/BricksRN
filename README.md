# react-bricks

一个运行在IOS,Android,Web Mobile 的组件库。风格参照xzLightApp

## 运行web

> npm start

http://localhost:8000/demo.html

## 运行android

> 打开模拟器 
> npm run android

## 运行ios

> 打开模拟器
> npm run ios
